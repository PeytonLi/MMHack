# @mmhack/web

Next.js App Router frontend for the employee workflow.

## Current Scope

- SKU selection
- Mobile camera preview and frame capture
- Analysis placeholder view wired to shared pricing rules
- Audit-history placeholder route

## Commands

- `pnpm --filter @mmhack/web dev`
- `pnpm --filter @mmhack/web build`
- `pnpm --filter @mmhack/web typecheck`
- `pnpm --filter @mmhack/web lint`
